Video Game Sales

This projects serves to be a practice of my recent learning of the plotly python extention, in which its interactive capability truly intrigued me. It uses a kaggle dataset displaying a list of games and their sale numbers from 1980 to 2018. It was also my first taste of making a html website.

Python Tools Used
- pandas
- plotly
- Jupyter Notebook
- HTML

Featuring Analysis such as:

- Amount of video game sales per year
- Which console and genre had the most sales

Future improvements 
- Include comparisons between regional sales
- Genre popularity over the years
